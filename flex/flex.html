Les Flex Box CSS (raccourci pour CSS Flexible Box Layout Module) sont un standard CSS3 de disposition des éléments dans une page web1. Ce standard permet d'avoir un design adaptatif à l'écran. Les éléments peuvent être réagencés selon la taille de l'écran.

En anglais, « flex » signifie « mettre en avant« de façon exagérée. Aussi, si un jeune vous dit que « tu n'trouves pas que je suis trop flex ? » cela veut simplement dire : « tu ne trouves pas que j'ai trop la classe ? ». Cette définition du flex est très utilisée dans le rap américain.

Le flex est une feuille plastique thermocollante que l'on peut découper à sa guise avec une machine de découpe ou tout simplement à la main selon le modèle de son choix. Il s'applique ensuite à chaud sur le tissu. Cela permet de customiser ses vêtements, accessoires…
